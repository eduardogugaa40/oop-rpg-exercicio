# oop-rpg-exercicio

Projeto simples desenvolvido em Java com o objetivo de treinar os conceitos fundamentais de Programação Orientada a Objetos, especialmente herança, construtores, uso de super e sobrescrita de métodos.

O sistema simula personagens de um RPG, onde existe uma classe base chamada Personagem e classes filhas como Mago e Guerreiro, que herdam seus atributos e comportamentos, mas implementam o método atacar de forma diferente.

No programa principal, os objetos são instanciados como Personagem, mas executam o comportamento real de suas classes específicas, demonstrando o funcionamento do polimorfismo na prática.

Esse projeto tem caráter puramente educacional e foi criado como parte do processo de aprendizado de POO em Java.
